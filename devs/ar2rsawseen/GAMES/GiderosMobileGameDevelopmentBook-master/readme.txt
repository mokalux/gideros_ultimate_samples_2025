Code and Assets for Gideros Mobile Game Development Book

Code and assets are divided by chapters

Each chapter contains the resulting project, just so you could try it out and see where you might have missed something

You should open them through Gideros Studio, by going to File -> Open Project

And you should use the images provided with the sample projects, when going through the book and trying all out yourself
More information about the book: http://giderosgamedevelopment.com
Get the book: http://www.packtpub.com/gideros-mobile-game-development/book
